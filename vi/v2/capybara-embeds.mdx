Nhánh này kiểm thử [hình ảnh và nội dung nhúng](https://www.mintlify.com/docs/create/image-embeds) của Mintlify trên **30** trang tiếng Anh.

**Mỗi trang được liên kết bên dưới đều dùng một bức ảnh capybara khác nhau** (các tệp cục bộ tại `/images/capybaras/unique-*.jpg`, mỗi tệp dưới 2MB). Các trang chuyển đổi sáng/tối dùng hai bức ảnh độc nhất.

## Kỹ thuật đã sử dụng

| Kiểu # | Kỹ thuật |
| --- | --- |
| 0 | Markdown + đường dẫn cục bộ `/images` |
| 1 | `<Frame>` + Markdown + đường dẫn cục bộ `/images` |
| 2 | HTML `<img>` + `style` inline có chiều rộng |
| 3 | HTML `<img>` + `noZoom` |
| 4 | Ảnh có liên kết (`<a>` + `<img noZoom>`) |
| 5 | Cặp ảnh cho chế độ sáng/tối (hai ảnh độc nhất) |
| 6 | `<Frame>` + HTML `<img className="rounded-xl">` |
| 7 | Markdown + tài sản cục bộ độc nhất |
| 8 | YouTube `<iframe>` trong `<Frame>` + ảnh tĩnh độc nhất |
| 9 | Markdown thuần + đường dẫn cục bộ `/images` |
