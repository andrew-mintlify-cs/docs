title: "Galerie d'images et d'embeds de capybaras"
description: "Index des pages qui présentent les images et embeds Mintlify avec des photos uniques de capybaras."
---

Cette branche teste les [images et embeds](https://www.mintlify.com/docs/create/image-embeds) de Mintlify sur **30** pages en anglais.

**Chaque page liée ci-dessous utilise une photo de capybara différente** (fichiers locaux dans `/images/capybaras/unique-*.jpg`, chacun inférieur à 2 Mo). Les pages avec bascule clair/sombre utilisent deux photos uniques.

## Techniques utilisées

| Style # | Technique |
| --- | --- |
| 0 | Markdown + chemin local `/images` |
| 1 | `<Frame>` + Markdown + chemin local `/images` |
| 2 | HTML `<img>` + `style` en ligne avec largeur |
| 3 | HTML `<img>` + `noZoom` |
| 4 | Image liée (`<a>` + `<img noZoom>`) |
| 5 | Paire d'images mode clair/sombre (deux photos uniques) |
| 6 | `<Frame>` + HTML `<img className="rounded-xl">` |
| 7 | Markdown + ressource locale unique |
| 8 | `<iframe>` YouTube dans `<Frame>` + miniature unique |
| 9 | Markdown simple + chemin local `/images` |
