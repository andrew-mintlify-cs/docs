> Langue active : **Français**. Le contenu pédagogique principal est ci-dessous.
Use **Tabs** when visitors frequently move between peer sections of one product.

Concrete examples on this site:
- **Install Guide** — setup content
- **API Reference** — endpoint docs
- **Cookbook** — recipes

Those three sections are peers. Tabs make that peer relationship obvious.
