Ця гілка тестує [зображення та вбудовування](https://www.mintlify.com/docs/create/image-embeds) Mintlify на **30** англомовних сторінках.

**Кожна сторінка за посиланням нижче використовує окреме фото капібари** (локальні файли за шляхом `/images/capybaras/unique-*.jpg`, кожен розміром до 2 МБ). Сторінки з перемиканням світла/темна використовують два унікальні фото.

## Використані техніки

| Стиль # | Техніка |
| --- | --- |
| 0 | Markdown + локальний шлях `/images` |
| 1 | `<Frame>` + Markdown + локальний шлях `/images` |
| 2 | HTML `<img>` + вбудований `style` з шириною |
| 3 | HTML `<img>` + `noZoom` |
| 4 | Зображення-посилання (`<a>` + `<img noZoom>`) |
| 5 | Пара зображень для світлої/темної теми (два унікальні фото) |
| 6 | `<Frame>` + HTML `<img className="rounded-xl">` |
| 7 | Markdown + унікальний локальний ресурс |
| 8 | YouTube `<iframe>` у `<Frame>` + унікальний кадр |
| 9 | Звичайний Markdown + локальний шлях `/images` |
