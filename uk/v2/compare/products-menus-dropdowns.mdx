Ці три можливості можуть здаватися "випадаючими меню", але вони розташовані в **різних місцях** інтерфейсу.

## 1. Products (перемикач продуктів)

- **Де:** перемикач продуктів (зазвичай угорі, біля логотипу)
- **Що перемикає:** цілі сайти документації продуктів у межах одного розгортання Mintlify
- **Ментальна модель:** Payments vs Identity vs Analytics — окремі продукти, окремі дерева документації

У цій демонстрації відкрийте **перемикач продуктів** і зверніть увагу на такі назви, як **Tabs Product**, **Dropdowns Product**, **Plain Product** та записи **Product + Menu**.

## 2. Product menus (`menu` у продукті)

- **Де:** також у **перемикачі продуктів**
- **Що змінюється:** замість переходу одразу до одного продукту, цей запис відкриває **багатостовпчикове меню** (Guides / API / Changelog / …)
- **Ментальна модель:** один продукт, кілька "вхідних дверей", які виводяться з перемикача

Спробуйте **Product + Menu: Core API** або **Product + Menu: Analytics** у перемикачі продуктів — ці записи розгортаються у меню з підписаними пунктами.

## 3. Dropdowns (шаблон навігації бічної панелі)

- **Де:** **вгорі лівої бічної панелі**, всередині продукту
- **Що перемикає:** розділи *в межах* поточного продукту (Docs / API Spec / Partners)
- **Ментальна модель:** як Tabs, але компактніше — сусідні розділи ховаються в меню бічної панелі замість горизонтального рядка вкладок

Відкрийте **Dropdowns Product**, а потім скористайтеся випадаючим меню в бічній панелі. Цей елемент керування **не** є перемикачем продуктів.

## Швидке порівняння

| Функція | Розташування в UI | Перемикає між | Ключ конфігурації |
| --- | --- | --- | --- |
| **Products** | Перемикач продуктів | Цілі продукти | `products[]` |
| **Product menus** | Перемикач продуктів (розгорнутий запис) | Вхідні точки *всередині* одного продукту | `products[].menu[]` |
| **Dropdowns** | Верх бічної панелі | Розділи в межах поточного продукту | `dropdowns[]` |

Якщо два елементи керування виглядають схоже, перевірте, **де** з'явилося меню: у перемикачі продуктів чи в бічній панелі.
