## Демонстрація вбудовування зображення

Заміна зображення для світлого/темного режиму. **Кожен режим використовує унікальне фото, яке не повторюється на інших сторінках галереї.**

- Світлий: `/images/capybaras/unique-26.jpg`
- Темний: `/images/capybaras/unique-33.jpg`
