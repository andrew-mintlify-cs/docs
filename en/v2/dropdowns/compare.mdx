This page lives inside the **Dropdowns Product**.

## What you should notice in the UI

1. Look at the **top of the sidebar** — there is a **dropdown menu**, not a row of horizontal tabs.
2. Open it. You should see three labeled items:
   - **Product Docs Dropdown**
   - **API Spec Dropdown**
   - **Partner Hub Dropdown**
3. Choosing one replaces the sidebar section content.

## Tabs vs Dropdowns

| | Tabs (other product) | Dropdowns (this product) |
| --- | --- | --- |
| Control location | Horizontal top navigation | Sidebar dropdown selector |
| Always-visible peers | Yes — sibling tabs stay on screen | No — peers stay inside the menu |
| Best for | A few peer sections switched often | More sections packed into less space |

Switch back to **Tabs Product** to see the horizontal tabs again.
