This branch tests Mintlify [images and embeds](https://www.mintlify.com/docs/create/image-embeds) on **30** English pages.

**Every linked page below uses a different capybara photo** (local files under `/images/capybaras/unique-*.jpg`, each under 2MB). Light/dark swap pages use two unique photos.

## Techniques used

| Style # | Technique |
| --- | --- |
| 0 | Markdown + local `/images` path |
| 1 | `<Frame>` + Markdown + local `/images` path |
| 2 | HTML `<img>` + inline `style` width |
| 3 | HTML `<img>` + `noZoom` |
| 4 | Linked image (`<a>` + `<img noZoom>`) |
| 5 | Light/dark mode image pair (two unique photos) |
| 6 | `<Frame>` + HTML `<img className="rounded-xl">` |
| 7 | Markdown + unique local asset |
| 8 | YouTube `<iframe>` in `<Frame>` + unique still |
| 9 | Plain Markdown + local `/images` path |
