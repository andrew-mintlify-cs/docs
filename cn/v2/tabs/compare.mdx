This page lives inside the **Tabs Product**.

## What you should notice in the UI

1. Look at the **horizontal tabs near the top** of the page (not the sidebar):
   - **Compare Tabs**
   - **Install Guide**
   - **API Reference**
   - **Cookbook**
2. Clicking a tab swaps the whole section and usually changes the URL path.
3. Tabs stay visible so you can jump between parallel sections for the same audience.

## Tabs vs Dropdowns (quick contrast)

| | Tabs | Dropdowns |
| --- | --- | --- |
| Where they appear | Horizontal bar across the top | Compact menu at the top of the sidebar |
| Best for | Parallel sections you switch often (Guides / API / Cookbook) | Many sections when you want a smaller chrome footprint |
| Visual cue | You always see sibling tabs | You open a menu to pick another section |

Now switch the product switcher to **Dropdowns Product** and compare: you will see a dropdown control instead of these top tabs.
