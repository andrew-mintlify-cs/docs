此分支在 **30** 个英文页面上测试 Mintlify 的[图片与嵌入](https://www.mintlify.com/docs/create/image-embeds)。

**下方列出的每个页面都使用不同的水豚照片**（本地文件位于 `/images/capybaras/unique-*.jpg`，每个小于 2MB）。明暗切换页面使用两张不同的独特照片。

## 使用的技巧

| 样式 # | 技巧 |
| --- | --- |
| 0 | Markdown + 本地 `/images` 路径 |
| 1 | `<Frame>` + Markdown + 本地 `/images` 路径 |
| 2 | HTML `<img>` + 内联 `style` 宽度 |
| 3 | HTML `<img>` + `noZoom` |
| 4 | 带链接的图片（`<a>` + `<img noZoom>`） |
| 5 | 明暗模式图片对（两张独特照片） |
| 6 | `<Frame>` + HTML `<img className="rounded-xl">` |
| 7 | Markdown + 独特的本地资源 |
| 8 | `<Frame>` 中的 YouTube `<iframe>` + 独特封面图 |
| 9 | 纯 Markdown + 本地 `/images` 路径 |
