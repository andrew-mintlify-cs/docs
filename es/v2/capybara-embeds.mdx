Esta rama prueba las [imágenes y embeds](https://www.mintlify.com/docs/create/image-embeds) de Mintlify en **30** páginas en inglés.

**Cada página enlazada abajo usa una foto de capibara diferente** (archivos locales en `/images/capybaras/unique-*.jpg`, cada uno menor a 2 MB). Las páginas con intercambio claro/oscuro usan dos fotos únicas.

## Técnicas utilizadas

| Estilo # | Técnica |
| --- | --- |
| 0 | Markdown + ruta local `/images` |
| 1 | `<Frame>` + Markdown + ruta local `/images` |
| 2 | HTML `<img>` + `style` en línea con ancho |
| 3 | HTML `<img>` + `noZoom` |
| 4 | Imagen enlazada (`<a>` + `<img noZoom>`) |
| 5 | Par de imágenes modo claro/oscuro (dos fotos únicas) |
| 6 | `<Frame>` + HTML `<img className="rounded-xl">` |
| 7 | Markdown + recurso local único |
| 8 | `<iframe>` de YouTube en `<Frame>` + miniatura única |
| 9 | Markdown simple + ruta local `/images` |
